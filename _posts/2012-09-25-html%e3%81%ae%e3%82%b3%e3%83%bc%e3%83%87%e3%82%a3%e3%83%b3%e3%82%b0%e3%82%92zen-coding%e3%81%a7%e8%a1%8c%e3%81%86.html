Zen-Codingというのがあります。
これはすごく強力なコードアシスト機能？で、ある決まった書式をHTMLやCSSに展開してくれるものです。
スクラッチでHTMLコーディングとかすると、divとか階層深くなりすぎるとちょっとうんざり…というのもあまりなくなるかも。

書式はHTMLやCSSをちょっと知っていればすぐ覚えられるようなものなのでこれは使ったほうがお得だし、ちょっと長いのが綺麗に決まると結構嬉しいので使いましょう！

テキストで書くより動画で見てみたほうが手っ取り早いので動画作成しました。
これは、Twitter-Bootstrapのformを書いてみたところです。

httpv://www.youtube.com/watch?v=ONOYowdfvy0

div.menu>ul>li*3

しかも親子関係の構造もずれにくくなるので安心。

今回はTextmate2でやったのでその機能拡張方法も書いておきます。
Textmate2は既に導入済みとして、zen-codingのBundleは下記にあります。

zen-coding -
Set of plugins for HTML and CSS hi-speed coding - Google Project Hosting
http://code.google.com/p/zen-coding/

上記URLに書式の例や詳細リファレンスへのリンクもあります。

今回は、Zen Coding for Textmate v0.7を使いました。これをダウンロードします。
Zen.Coding-TextMate.v0.7.zip - zen-coding -　Zen Coding for TextMate v0.7 - Set of plugins for HTML and CSS hi-speed coding - Google Project Hosting
http://code.google.com/p/zen-coding/downloads/detail?name=Zen.Coding-TextMate.v0.7.zip

ダウンロードしたzipファイルの中に「Zen Coding.tmbundle」というファイルがあるので、それをTextmateのアイコン(Dockに入っているやつでも良いです）にドラッグドロップするとすぐ使えるようになります。（あと、本当はTextmate2のBundle Supportを有効にしないとダメかも）

お勧めです。
