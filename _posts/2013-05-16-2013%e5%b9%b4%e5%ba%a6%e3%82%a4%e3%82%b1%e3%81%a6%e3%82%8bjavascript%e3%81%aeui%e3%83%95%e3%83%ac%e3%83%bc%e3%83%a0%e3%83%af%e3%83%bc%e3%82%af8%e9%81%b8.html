JQueryUIやTwitter Bootstrapに食傷気味なので色々探してみた個人的に興味を持ったJavaScriptのUIFrameworkのリストです。
jQueryUIやBootstrapの他、HTML Kickstartも有名なので除外します。
また、cssだけのも基本的に含めません。あと、無料のやつ。だからKendoUIも除外。
色使いはデフォルトの設定の印象で、大概デモを見ての感想です。

Gumbyframework

これは自分が知らなかっただけでもしかして既に有名なのかもしれない。完成度高すぎ。
Twitter Bootstrapのようであるが、その不足部分も補ってくれているような印象。（グリッドレイアウトのセンタリングとか）
完成度高い！と思うのはデザインの小洒落度もある。
Boostrapじゃない縛りとかであればこれを使ってみるのも良いかも。
ライセンスはMIT。

Gumby - A Flexible, Responsive CSS Framework - Powered by Sass
http://gumbyframework.com/

jQuery TOOLS

jQueryUIを補完するようなもののようである。
使用画像の光沢が美しいので好感度は良い。jQueryUIとくらべて目立ったのはScrollableとRangeInput, Exposeあたりが使いたいなと思った。
ライセンスはGPL3だと思う。
jQuery TOOLS - The missing UI library for the Web
http://jquerytools.org/

Tuktuk

シンプルで最低限のものしかないけど丁寧にやってある印象。ナビゲーションのエントリ等じんわり表示されていたり細かいこだわりが見られる職人芸ぽい感じである。
アイコンも多数、FontAwesomeをベースにしたWebフォントを使っている。
ライセンスはGPLv3ベースのもののようである。

Tuktuk - Simple (but powerful) RWD Framework
http://tuktuk.tapquo.com/

AlloyUI

jQueryベースではなく、YUI3のライブラリとのことだが、コンポーネントが豊富である。（Windowsのエクスプローラのようなツリービューもある。）
個人的にはDiagram Builderにちょっとビックリした。それ専用のライブラリなら簡単かもしれないが…。
あまりYUIに詳しくないのだが、jQueryと併用してあまり問題でないもんかな？
ライセンスはBSD。
AlloyUI
http://alloyui.com/

Metro UI CSS

BootstrapにもMetroのテーマはあったと思うが、こちらの方のがよりリアルにWindows8のUIを再現？しているように思う。
特にカルーセルそう思った。
ライセンスはMITだと思う。
Metro UI CSS
http://metroui.org.ua/

FuelUX

Bootstrapをベースにしたもののようであるが、DataGridとWizardが目立ったコンポーネントでした。
FuelPHPとなんか関係あるのかと思ったけど特に関係ないみたい。
ライセンスはMIT
Fuel UX
http://exacttarget.github.io/fuelux/

GroundworkCSS

デフォルトのエメラルドグリーンの色使いが独特である。
グラデーションは薄く、MetroでBootstrapのような角丸的な印象です。
Tilesがちょっといいかなと思ったが、これは普通の正方形のボタン用かな？
GroundworkCSS ♥ A Responsive HTML5, CSS & Javascript Framework
http://groundwork.sidereel.com/

jquery-jkit

コンポーネントのデザインとしてはちょっとアレだが、どうもdata-属性を使って色々やるタイプのようである。
なるべくDOM側に設定を持たせたい場合なんかはいいんじゃないかな？
ライセンスはMIT。

jKit - jQuery based UI Toolkit - Index
http://jquery-jkit.com/

商用で気になったのもあった。Zino UI、jqWidgetとか。
まだまだたくさんありそうだけど、なんとなくSencha Touchのようなレイアウトがギュッと凝縮されている感じのよりも
ドーンとレイアウトされているのが個人的には好きです。
実際使ってみるとまた細かいところの違いがわかるのだと思うのですがそこまでは出来ていません。
いぶし銀なところでtuktukを使ってみたいなぁと思ってます。
