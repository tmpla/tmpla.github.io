WindowsでExcelを使うのに慣れている場合、あれ、WindowsだとこうだけどMacのだとどうなんだろ？
とよく思ったのがあって、それの温めていたネタを書きます。
これは3ヶ月我慢して自然に出てきたやつなのでそれなりに需要あるはず。

日付関係

WindowsだとCtrl + ;, Ctrl + :で大丈夫だが、Macで言うCtrlはCommandキーのような気がするのでごっちゃになる。

本日日付

Ctrl + ;

現在時刻

Command + :

セル編集関係

セル内編集

Windowsならばセルを選択状態でF2を押すとセルの文字列を編集できるがMacならこれ。
Ctrl + U

セル内折り返し

Windowsならばセル編集中にAlt + Returnでやってたがこれ。
Option + Command + Return

選択範囲クリア

WindowsならBackspaceで消えたような気が（あるいは右クリック→クリアだったかな？）するがこれ。
Ctrl + b

行関係

行挿入

Wind（略）Ctrl + [+]だったと思う。
Ctrl + i

行削除

W（ｒｙならCtrl + [-]だったと思う。
Ctrl + k

エディタ使いには釈迦に説法だが、最初は面倒臭いと思っても、ショートカットは覚えたほうが良い。
一度の作業がマウス作業等からショートカットに変えて１秒短縮出来れば、一万回なら一万秒≒167分≒2.7時間位お得で時給800円ならば2160円位になる。
覚えて店長に相談しよう！
