gitのコマンドの話。

環境はMacのXCodeのCommand Line Toolsが入っている状態で、gitのバージョンは下記です。

$ git --version
git version 1.7.7.5 (Apple Git-26)

管理ファイルを追加する

あるディレクトリ以下のファイルを全部追加する場合は下記手順でやります。主に自分だけでバージョン管理するとき用じゃないかな。

$ cd project
$ git init
$ git add .
$ git commit

git add . でピリオド（カレントディレクトリ）を指定しています。
git commit時にはコメントを入力しないとコミットされません。コメントは推奨は

サマリ
(空白行）
詳細

の様に書くのが良いそうです。

例えば~/repo/test-file
という感じでrepoディレクトリ配下にtest-fileというファイルがあるだけの場合に、「git status」をいちいち発行してみた場合の例です。

$ touch repo/test-file
$ cd repo/
$ git status
