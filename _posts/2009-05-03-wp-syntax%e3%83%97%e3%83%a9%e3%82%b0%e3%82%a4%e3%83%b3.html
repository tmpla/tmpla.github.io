このブログはWordpressで作成しているわけですが、自分の場合、よくソースコードを引用するので、そこだけスタイルシートを指定して（たとえば黒地にするとか）表示させたいなと思っていました。

たったさっき、どうしてもそういう風に書きたい欲が高まったので、他の人はどうやってるんだろう？と調べて、プラグインを発見しました。それがWP-Syntaxプラグインです。

このプラグインはGeSHi - Generic Syntax Highlighterというのの拡張なのでしょうか？とにかくサポートしている言語の数が多い。

たとえば、javaのソースコードを記載する場合は以下のように書きます。

<pre lang="java">
public static final void main(String[] args){
    System.out.println("Hello World!");
} 
</pre>

以下が変換された結果です。

public static final void main(String[] args){
    System.out.println("Hello World!");
}
