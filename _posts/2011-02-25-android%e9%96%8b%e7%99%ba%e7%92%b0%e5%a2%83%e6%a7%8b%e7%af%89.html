iPhoneアプリはMacがないとダメっぽいので、Androidの開発環境を作ってみようじゃないか、という話。

環境は

・Windows Vista Home SP1
で、あとはJDKを入れときましょう。JREじゃなくてJDKが必要です。
自分の環境はこんな感じでした。

>java -version
java version "1.6.0_22"
Java(TM) SE Runtime Environment (build 1.6.0_22-b04)
Java HotSpot(TM) Client VM (build 17.1-b03, mixed mode, sharing)

まずはじめにEclipseをダウンロードします。いろいろプラグインも導入済みなので、pleiadesの最新版（現在はHERIOS）のJava開発版でいいと思います。

http://mergedoc.sourceforge.jp/

ダウンロードして解凍したら、「Android開発用」とかの名前でショートカットを作っておくと良いと思います。
というのは、自分のマシンにはすでに普通のJavaEE開発用と、PDTが入っているのでまた別の環境として使用したいので。

Android SDKのセットアップ

そしたら今度はAndroid SDKをダウンロードしてインストールしておきます。

http://developer.android.com/sdk/index.html

から「installer_r08-windows.exe」を選んでインストールします。（別にzipファイルを選んで解凍してもよろしいのではないかと）

インストール完了して、SDKマネージャを起動するチェックボックスにチェックが入っていると（デフォルトで入っている）SDKマネージャ
なるものが起動します。

左側メニューの「Available packages」を選択すると、右側ウインドウに選択できるSDKやアドオンが表示されるので、
ダウンロード速度とディスク容量を気にしなければとりあえずは「Android Repository」は全部、「Third Party Add-ons」はGoogle Inc. add-onsを選択して「Install Selected」ボタンを押下します。
すると、ライセンスに同意する的な画面が出るので、

同意してください。すべて同意するなら「accept-all」のラジオボタンが便利です。
その後「Install」ボタンを押下するとインストールが始まります。

AVDの作成

インストールが完了したら、開発用のエミュレータ環境（Virtual Device）を作成します。

右側の「New」ボタンを押して作成します。
今回は、Android2.2用として作成してみます。
なので、各項目に下記のように入力します。
Name:Android2.2
Target:Android2.2 - API Level 8
SD Card: size 64 MiB

これで「Create AVD」ボタンを押して下記のようになればSDKとAVDの環境構築は完了です。
