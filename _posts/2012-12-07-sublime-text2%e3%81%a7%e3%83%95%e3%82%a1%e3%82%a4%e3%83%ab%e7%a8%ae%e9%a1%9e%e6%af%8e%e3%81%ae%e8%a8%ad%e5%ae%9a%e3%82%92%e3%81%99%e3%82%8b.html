Sublime Text 2でタブサイズとかの設定をするにはPreferences - Settings - UserからPreferences.sublime-settingsを設定すれば良い。

ただ、ファイルの種類毎にこの設定を変えたい時がよくあるが、どうやんのかわからなかったのでちょっと調べてみた。
例えば、coffeesciptやhamlならばインデントはタブにしたいが、yamlならばスペースにしたい、と言うことがあったので。
この場合は、yamlだけタブをスペース2つのレイアウトでタブをスペースに変換する設定にしておけば良いはず。

そのやり方は下記に書いてあった。

Settings - Sublime Text 2 Documentation
http://www.sublimetext.com/docs/2/settings.html

この場合ならばyamlの設定を作れば良さそうだ。

{
    "tab_size": 2,
    "translate_tabs_to_spaces": true
}

と言うファイルを
（どっか）/Sublime Text 2/Packages/User/YAML.sublime-settings
と言う名前で保存する。Preferences - Browse PackagesでそのあたりのディレクトリがFinderで開くので参考にしよう。

それだけでもう適用されているようでした。
