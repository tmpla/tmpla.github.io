ショートカット

実は漢字Talk（おぉ一発で変換できた）7.5位からMacOS9あたりまでは結構使っていたのですが、
その頃からショートカットを多用する文化？だったと思います。
今でもそれはそのようで、ショートカットはいろんなアプリケーションのメニューの横に記載があるのでだいたいわかるのですが、
Command, Option, Control, Shiftの記号がどれがどれなんだかわからんというのがあります。
これは下記ページで調べました。

Mac OS X のキーボードショートカット

CommandキーはAirの場合はキーボードにマークが書いてあるのでわかるのですが、他がわからないので、Shiftは上矢印、Controlはべき乗、Optionはエスカレーターと覚えときます。

Windowの操作になれている場合だと、Ctrlボタンとの組み合わせが結構多かったと思うのですが、WindowsのCtrlボタンはほぼCommandキーになりますね。

日本語変換

日本語打つとき、スペースキーの右左にある「英数」「かな」で切り替えるのだと思うのですが、かなモードで英数打っちゃったりあるいは逆の場合には打ち直すしかないのかと思ってました。
Windowsであれば、F7からF11キーくらいまでのファンクションキーで再変換出来たのですが、Macでは出来ないようで参りましたが、
fnキー＋ファンクションキーでWindowsと同じように出来るようでした。
しかしなれないうちは変換に「１」キーを押しちゃう（Windowsの漢字キー）ので修行が必要だな・・・。
